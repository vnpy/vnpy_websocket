## 说明

基于asyncio和aiohttp开发的协程异步Websocket API客户端，用于开发高性能的Websocket交易接口。

## 安装

安装环境推荐基于3.7.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。

直接使用pip命令：

```
pip install vnpy_websocket
```

下载解压后在cmd中运行

```
pip install .
```
