## 说明

基于asyncio和aiohttp开发的协程异步Websocket API客户端，用于开发高性能的数字货币交易接口。

## 安装

安装需要基于2.6.0版本以上的[VN Studio](https://www.vnpy.com)。

直接使用pip命令：

```
pip install vnpy_websocket
```

下载解压后在cmd中运行

```
python setup.py install
```
